## 🎯 О проекте

**ChessWPF** — это профессиональная шахматная программа, написанная на **C# / WPF (.NET 8)**.
Проект создан как полноценное настольное приложение с современным интерфейсом,
умным ИИ-противником, сетевой игрой, личным кабинетом и анализом партий как на Chess.com.

> 🇷🇺 Интерфейс полностью на русском языке

## 🤖 ИИ противник

Искусственный интеллект реализован на алгоритме **Minimax с Alpha-Beta отсечением**.

```
         Позиция
        /        \
     Ход1        Ход2          ← Максимизирующий (ИИ)
    /    \      /    \
  X1     X2  X3     X4        ← Минимизирующий (игрок)
  ✂️          ✂️               ← Alpha-Beta обрезает ветки
```

### Оценка фигур

| Фигура | Символ | Очки |
|--------|--------|------|
| Пешка | ♙ | 100 |
| Конь | ♘ | 320 |
| Слон | ♗ | 330 |
| Ладья | ♖ | 500 |
| Ферзь | ♕ | 900 |
| Король | ♔ | 20 000 |

### Дополнительная оценка

Помимо материала, ИИ учитывает **позиционные таблицы** для каждой фигуры:

- 📍 Пешки стремятся к центру и продвижению
- 🐴 Кони выгоднее в центре доски
- 👁️ Слоны получают бонус за длинные диагонали
- 🏰 Ладьи на открытых вертикалях
- 👑 Король прячется в дебюте и миттельшпиле

## 🌐 Сетевая игра

Игра по локальной сети через **TCP сокеты** (порт **5555**).

```
┌─────────────────────┐         ┌─────────────────────┐
│   ХОСТ (Белые)      │         │   КЛИЕНТ (Чёрные)   │
│                     │         │                     │
│ 1. Нажать "Хост"    │◄───────►│ 1. Нажать "Клиент"  │
│ 2. Сообщить IP      │  TCP    │ 2. Ввести IP хоста  │
│ 3. Ждать соперника  │ :5555   │ 3. Подключиться     │
│ 4. Играть белыми    │         │ 4. Играть чёрными   │
└─────────────────────┘         └─────────────────────┘
```

> 💡 Узнать свой IP: `Пуск → cmd → ipconfig → IPv4`

## 👤 Личный кабинет

Полноценный профиль игрока сохраняется между сессиями в `%AppData%\ChessWPF`.

| Функция | Описание |
|---------|----------|
| Никнейм | Редактируется в профиле |
| Аватарка | Любое изображение с диска |
| Статистика | Победы / поражения / ничьи / маты / паты / серии / % побед |
| Звания | Новичок → Любитель → Клубный → Кандидат → Мастер → Гроссмейстер |
| Скины доски | 8 тем: Классика, Зелёная, Синяя, Розовая, Фиолетовая, Серая, Ночная, Золотая |

### Система званий

| Звание | Условие |
|--------|---------|
| 🔰 Новичок | Старт |
| ⚔️ Любитель | 5 побед |
| 🛡️ Клубный | 15 побед |
| 🎯 Кандидат | 30 побед |
| 🏅 Мастер | 50 побед |
| 👑 Гроссмейстер | 100 побед |

## 🔍 История и анализ партий

### История
- Автоматически сохраняются последние **10 партий**
- Отображается: результат, дата, количество ходов, режим игры
- Кнопка **🔍 Анализ** у каждой записи

### Анализ партии

Работает аналогично анализу на **Chess.com** — разбор каждого хода с оценкой:

| Оценка | Символ | Значение |
|--------|--------|----------|
| Отличный ход | `!!` | Лучший возможный ход |
| Хороший ход | `!` | Сильный ход |
| Неточность | `?!` | Небольшая потеря преимущества |
| Ошибка | `?` | Значительная потеря |
| Зевок | `??` | Грубая ошибка |

- Перемотка ходов: **⏮ ◀ ▶ ⏭**
- Оценочная шкала позиции в реальном времени
- Подсказка лучшего хода от ИИ
- Кликабельный список ходов с иконками качества

## 🧩 Головоломки

8 встроенных задач для тренировки тактики:

| Название | Тип | Сложность |
|----------|-----|-----------|
| Мат в 1 ход #1–3 | Финальная атака | Лёгкая |
| Мат Дурака | Быстрый мат | Лёгкая |
| Вилка конём | Тактика | Средняя |
| Вилка пешкой | Тактика | Лёгкая |
| Спёртый мат | Тактика | Средняя |
| Двойной шах | Комбинация | Сложная |

- 💡 Система подсказок с подсветкой нужной фигуры
- Индикатор прогресса `1 / 8`
- Переход между задачами и сброс позиции

## 🏗 Архитектура

```
ChessWPF/
│
├── 📁 Views/                   # Представления (UI)
│   ├── MainMenuView.xaml       # Главное меню
│   ├── GameView.xaml           # Игровой экран
│   ├── ProfileView.xaml        # Личный кабинет
│   ├── HistoryView.xaml        # История партий
│   ├── AnalysisView.xaml       # Анализ партии
│   ├── PuzzleView.xaml         # Головоломки
│   ├── VictoryOverlay.xaml     # Конфетти при победе
│   ├── PromotionDialog.xaml    # Превращение пешки
│   ├── GameOverDialog.xaml     # Конец игры
│   ├── TimeDialog.xaml         # Выбор времени
│   ├── DifficultyDialog.xaml   # Выбор сложности
│   └── LanSetupDialog.xaml     # Настройка сети
│
├── 📁 Models/                  # Модели данных
│   ├── Board.cs                # Шахматная доска 8×8 + LoadFEN
│   ├── Piece.cs                # Фигура (тип, цвет, флаги)
│   ├── Move.cs                 # Ход (откуда, куда, флаги)
│   ├── PlayerProfile.cs        # Профиль + скины + singleton
│   ├── GameRecord.cs           # История партий + анализ
│   ├── AppTheme.cs             # Темы приложения
│   ├── PuzzleData.cs           # Встроенные головоломки
│   ├── PieceType.cs            # Перечисление типов фигур
│   └── PieceColor.cs           # Перечисление цветов
│
├── 📁 GameLogic/               # Игровая логика
│   ├── GameManager.cs          # Управление состоянием игры
│   ├── MoveGenerator.cs        # Генерация псевдо-легальных ходов
│   └── MoveValidator.cs        # Фильтрация нелегальных ходов
│
├── 📁 AI/                      # Искусственный интеллект
│   └── AIManager.cs            # Minimax + Alpha-Beta + PST
│
├── 📁 Network/                 # Сетевая игра
│   ├── LANServer.cs            # TCP сервер (хост)
│   └── LANClient.cs            # TCP клиент
│
├── 📁 UI/                      # UI утилиты
│   ├── PieceImages.cs          # Unicode символы фигур
│   └── BoardRenderer.cs        # Геометрия доски
│
└── 📁 PythonAI/                # Python движок (опционально)
    └── chess_engine.py         # Minimax на Python + TCP сервер
```
